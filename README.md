## GoStack - Rocketseat Bootcamp - Challange 06

E-commerce mobile app created using React Native, Redux and Redux Saga. <br />

Instructions of the challange can be found here: https://github.com/Rocketseat/bootcamp-gostack-desafio-07/blob/master/README.md

## Instructions

- Run "yarn" command to install all dependencies.
- Application requires JSON Server installed (you can install it via "yarn global add json-server"). Once this is installed, you can run "json-server server.json --host hostname -p 3333".
- Run "react-native run-android" to build and launch the app.
